#### 스택의 push 알고리즘: append 메소드를 통해 리스트의 마지막에 데이터 삽입

```
def push(item):
    s.append(item)
```

- 단지, append는 속도가 느리다. 지금은 그냥 개념적으로 사용하는 정도.

- 스택의 구현

```
def push(item, size):
    global top
    top +=1
    if top == size:
        print('overflow!')
    else:
        stack[top] = item
size = 10
stack = [0]*size
top = -1

push(10, size)
top += 1
stack[top] = 20
```

#### 스택의 pop 알고리즘

- 스택의 구현

```
def pop():
    global top
    if top == -1
        print('underflow')
        return 0
    else:
        top -= 1
        return stack[top+1]
print(pop())

if top > -1:
    top -= 1
    print(stack[top+1])
```
